User Stories from Product Backlog that we will address in this sprint:
a) As a user I should be able to save the status of the program						
b) As a user I should be able to load the saved state of the program
c) As a user I should be able to remove a directory from the file system
d) As a user I should be able to find all the directories of a given name
e) As a user I should be able to redirect output from all commands to a file 							
f) As a user I should be able to make multiple directories at once					
g) As a user I should be able to see the content of a directory and the content of all levels of nested directories within it
h) As a user I should be able to relocate a path from one path to another			
i) As a user I should be able to copy contents of one path to a new path					
j) As a user I should be able to fetch files from a URL to the file system
